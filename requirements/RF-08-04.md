# Requirement RF-08-04

### ID:
RF-08-04

### Type / Category:
Funcional - Personalització de plats

### Versió:
1.0

### Title:
Xat privat entre cuiner i client

### Descripció:
Quan un cuiner accepta una proposta de plat, el sistema ha de permetre la creació d'un xat privat entre el client i el cuiner per facilitar la comunicació sobre la comanda.
