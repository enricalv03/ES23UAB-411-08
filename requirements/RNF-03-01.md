# Requirement RNF-03-01

### ID:
RNF-03-01

### Type/Category:
Funcional — Seguiment de l'enviament

### Version:
1.0

### Title:
API del mapa

### Description:
La interfície utilitzada per mostrar la posició del repartidor en el mapa, haurà de utilitzar la API de Google Maps. El càlcul de la distància física entre client i cuiner ha de basar-se en les coordenades geogràfiques emmagatzemades als perfils. No es pot mostrar el llistat sense dades de localització del client.
