# Requirement RNF-D-02

### ID:
RNF-D-02

### Type/Category:
No funcional — Disseny / Seguretat

### Version:
1.0

### Title:
Seguretat i confidencialitat de les dades dels usuaris.

### Description:
El sistema ha de garantir la seguretat i confidencialitat de tota la informació dels usuaris: les contrasenyes s'emmagatzemin sempre xifrades i mai en text pla; les dades sensibles (adreces, restriccions alimentàries) es transmetin exclusivament per canals segurs (HTTPS/TLS); i les dades bancàries dels cuiners i repartidors es guardin de forma encriptada dins del sistema.
